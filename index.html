ServPro

The ServPro project was developed at Metaveiro, my first experience in the job market producing projects for clients.

The project was intended to be a business management Web App.

As this was my first and only big project at a professional level I was able to learn a lot, most of the technologies used in this project were unknown to me so I had to put a lot of effort and dedication into the project to learn these technologies. This was only possible due to my internship monitor who gave me a lot of support and knowledge throughout my year and a half at Metaveiro.

The Web App was developed in ASP.Net, that is, C#, HTML, CSS and JQuery were used. The database was developed in Microsoft SQL Server and the Entity Framework (EF) was also used to interact with the database.
